# Brand Profit Simulator

中古ブランドせどり向けの利益シミュレーターです。

- `index.html`: PC版とスマホ版の入口
- `desktop.html`: PC向けの一覧比較画面
- `mobile.html`: スマホ向けの店頭確認画面

販売価格帯と仕入れ価格帯から、実利益、売上利益率、ROI、仕入れ上限、判定を計算します。
結果は既存のGoogleスプレッドシートへ貼り付けやすいTSV形式でコピーできます。

## Google Sheetsへ転記する設定

`ブランドせどり_リサーチ記録_最新` の `利益計算 ` シートへ直接転記する場合は、Google Apps ScriptをWebアプリとして公開します。

1. Google DriveでApps Scriptを作成する
2. `gas/Code.gs` の中身を貼り付ける
3. `TOKEN` を自分だけの合言葉に変更する
4. デプロイ > 新しいデプロイ > ウェブアプリ
5. 実行ユーザー: 自分、アクセスできるユーザー: 全員
6. 発行されたWebアプリURLをシミュレーターの `Apps Script URL` に入れる
7. `TOKEN` と同じ合言葉をシミュレーターの `合言葉` に入れる

転記ボタンは、シートのA〜M列へ現在の計算結果を追記します。対象タブ名は末尾スペース込みの `利益計算 ` です。
